# portfolio

Static portfolio site (dark theme). Open [index.html](index.html) locally to preview the homepage.

Stack: Plain HTML, CSS, JS. Fonts: Inter (Google Fonts). Accent color: Cyan (#06B6D4).

To preview locally (recommended: VS Code Live Server or a simple local server):
```bash
python -m http.server 8000
# then open http://localhost:8000/index.html
```
