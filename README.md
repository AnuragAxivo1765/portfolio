# portfolio

Static portfolio site (dark theme). Preview via a local HTTP server (recommended) — opening via `file://` can break the DarkVeil module imports in some browsers.

Stack: Plain HTML, CSS, JS. Fonts: Inter (Google Fonts). Accent color: Cyan (#06B6D4).

To preview locally (recommended: VS Code Live Server or a simple local server):
```bash
python -m http.server 5173
# then open http://localhost:5173/index.html
```
